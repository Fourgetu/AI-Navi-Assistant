# 📘 **AI 对话快速导航助手 (AI Navi Assistant)** `v1.2.1`

### 📌 **插件简介**
这是一款专为 **Gemini** 和 **ChatGPT** 用户打造的**沉浸式增强插件**。它通过在页面右侧生成一个**智能导航栏**，帮助你在长篇累牍的 AI 对话中**瞬间定位**到任意一次提问，极大地提升了信息回溯效率。

---

### ✨ **核心功能**

* 🚀 **快速跳转**：点击右侧导航点，页面**平滑滚动**至对应提问位置。
* 🧠 **智能去重**：自动识别 Gemini 中的 `user-query-bubble`，确保大段 **YAML** 或代码输入**只占用一个导航位**，拒绝“刷屏”。
* 🎨 **沉浸式 UI**：
    * **动态悬停**：平时仅显示极简线条，鼠标靠近时自动展开**详细标题**。
    * **左对齐设计**：符合中文阅读习惯，文字排版**清爽不凌乱**。
* 📱 **智能适配**：
    * 支持**垂直滚动 (Scrollable)**，对话再长也能找全所有记录。
    * **底部边界优化**：确保点击**最后一个提问**也能精准定位，不被输入框遮挡。
* 🌐 **双平台支持**：完美兼容 `gemini.google.com` 和 `chatgpt.com`。

---

### 🛠️ **安装指南 (开发者模式)**
由于插件尚未提交至 Chrome 应用商店，请按照以下步骤手动安装：

1.  **下载代码**：从 GitHub 下载本项目压缩包并解压。
2.  **打开扩展程序**：在 Chrome 地址栏输入 `chrome://extensions/`。
3.  **开启开发者模式**：打开右上角的 **“开发者模式” (Developer mode)** 开关。
4.  **加载插件**：点击 **“加载已解压的扩展程序” (Load unpacked)**，选择你解压后的文件夹。
5.  **固定插件**：点击浏览器右上角的“拼图”图标，将 **AI 对话快速导航助手** 固定在工具栏上。

---

### 📂 **文件结构说明**

* **`manifest.json`**：插件的核心配置文件，定义权限与图标。
* **`content.js`**：核心逻辑脚本，负责 **DOM 监听**、**容器去重**与**平滑滚动跳转**。
* **`style.css`**：定义导航栏的视觉样式，包括**毛玻璃效果**与**美化滚动条**。
* **`icon.png`**：精美的**蓝底 AI Navi 图标**。

---

### ⚖️ **风险提示与声明**

* **数据安全**：本插件**完全运行在本地**，不会上传任何对话内容至第三方服务器。
* **私钥保护**：建议妥善保管打包生成的 `.pem` 私钥文件，**请勿**将其上传至公开仓库。
